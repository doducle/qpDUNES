qpDUNES
=======

qpDUNES - a DUal NEwton Strategy for convex quadratic programming


Basic instructions:
-------------------

1) Run   
  `make`
to compile.

2) Run
  `./examples/example1`
to test your successful installation.

Please consult
  [https://github.com/jfrasch/qpDUNES/wiki](https://github.com/jfrasch/qpDUNES/wiki)
for further information.
